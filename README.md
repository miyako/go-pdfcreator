Install latest [PDFCreator](https://docs.pdfforge.org/pdfcreator/en/pdfcreator/installing-pdfcreator/)

Install [ASP.NET Core Runtime 8](https://dotnet.microsoft.com/en-us/download/dotnet/8.0)
